CSS Frameworks & Styling

🎨

Utility-First CSS

Tailwind CSS, UnoCSS, Windi CSS. Rapid prototyping, design systems, and maintainable stylesheets.

🧱

Component Libraries

Material-UI, Ant Design, Chakra UI, Bootstrap, Bulma, Foundation. Pre-built components and design systems.

💅

CSS-in-JS

Styled-components, Emotion, JSS. Dynamic styling, theme providers, and component-scoped styles.

State Management Solutions

🔄

Redux Ecosystem

Redux Toolkit, React-Redux, Redux Saga, Redux Thunk. Predictable state management for complex applications.

🐻

Zustand

Lightweight state management. Simple API, TypeScript support, and minimal boilerplate for React apps.

🔮

MobX

Reactive state management. Observable data, computed values, and automatic UI updates through reactions.

⚛️

React Context

Built-in state sharing. Context API, useReducer hook, and component composition for state management.

📊

Apollo Client

GraphQL state management. Caching, optimistic updates, and seamless integration with GraphQL APIs.

🎪

Jotai

Atomic state management. Bottom-up approach, fine-grained reactivity, and flexible state composition.
